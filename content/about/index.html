Brad Parbs is a Milwaukee-based WordPress developer & engineer, focusing on high perfomance, user friendly, enterprise scale, API driven websites & applications.

Brad Parbs is currently a Senior Software Developer at Newfold Digital. Previously Nexcess, Awesome Motive working on WPForms, WebDevStudios, and Human Made.

Brad Parbs likes to build many things online. Check out his projects to see them.

Calling him a WordPress fanatic is putting it lightly. Brad Parbs has contributed to WordPress Core, built a variety of custom plugins & themes, as well as a variety of tools and guides. Building WordPress Plugins, Using the Command Line, WordPress Development, Responsive Web Design, Version Control, and more.
